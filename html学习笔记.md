## 基本的HTML页面

### 最简单的HTML页面

``` html
<!DOCTYPE html>
hello world!!!
```
### 一个HTML页面模板

``` html
<!DOCTYPE html>
<html lang="en">
    <head>
        <meta charset="utf-8">
    </head>
    <body>
        This is the body of the document.
    </body>
</html>
```

# 结构性的文本

``` html
<!DOCTYPE html>
<html lang="en">
    <head>
        <meta charset="utf-8">
    </head>
    <body>
        <h1>This is a top level heading</h1>
        <h2>This is a second level heading</h2>
        <h3>This is a third level heading</h3>
    </body>
</html>
```
